# 16 - Testes de Software

Resumo prático sobre tipos de testes e como aplicá-los.

Tipos principais
- Unit: testa unidades isoladas (funções, classes).
- Integration: testa integração entre módulos (DB, APIs).
- End-to-End (E2E): testa o sistema completo do ponto de vista do usuário.

Boas práticas
- Escreva testes automatizados para lógica crítica.
- Mantenha testes rápidos e independentes.

Detalhes sobre ferramentas e exemplos práticos foram movidos para `16.3 - ferramentas.md` e `16.2 - Estratégias de Teste` para facilitar leitura dirigida.
