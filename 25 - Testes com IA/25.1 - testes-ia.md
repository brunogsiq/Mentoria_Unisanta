# 25 - Testes com IA

Aplicações práticas de IA para melhorar a qualidade de testes.

Casos de uso
- Geração de cenários: criar variações de entrada automaticamente.
- Geração de massas de dados: dados realistas para testes sem expor PII.
- Análise de falhas: identificar padrões em logs e tendências de bugs.

Cuidados
- Valide saídas geradas por IA; não assuma correção automática.

Exemplo simples
- Usar um modelo para gerar casos de teste adicionais a partir de um caso base e revisar manualmente antes de integrar.
