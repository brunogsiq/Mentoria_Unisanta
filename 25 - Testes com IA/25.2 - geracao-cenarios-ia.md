# 25.2 - Geração de Cenários com IA

Ideias de aplicação de IA em testes:

- Gerar cenários de teste a partir de requisitos.
- Criar dados de teste sintéticos e variados.
- Analisar logs para identificar padrões de falha.

Exercício: usar uma API de LLM para gerar 10 casos de teste a partir de uma história.
