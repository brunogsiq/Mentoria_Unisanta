# 11 - Mobile

Noções práticas sobre desenvolvimento mobile.

Opções
- Nativo: Kotlin (Android), Swift (iOS) — melhor performance.
- Híbrido/Multiplataforma: React Native, Flutter — produtividade e reuso.

Exemplo de decisão
- App com alta integração com hardware → prefira nativo.
- App com entrega rápida para Android+iOS → considere Flutter/React Native.

Boas práticas
- Teste em emuladores e dispositivos reais.
- Gerencie permissões com cuidado (localização, câmera).
