# 21.2 - Logs e Estratégias de Debug

Boas práticas de logging e debug:

- Níveis de log (DEBUG, INFO, WARN, ERROR).
- Correlacionar requisições com request-id para rastrear erros.
- Ferramentas: debuggers do IDE, log aggregation (ELK, Grafana Loki).

Exercício: instrumentar logs em um endpoint e reproduzir um erro para análise.
