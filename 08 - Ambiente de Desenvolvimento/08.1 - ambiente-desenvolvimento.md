# 08 - Ambiente de Desenvolvimento

Configuração prática do ambiente para começar a desenvolver.

IDEs e ferramentas
- VS Code: leve e extensível (extensões para lint, debug, git).
- IntelliJ/PyCharm: IDEs completas para Java/Python.

Gerenciadores de pacote
- `npm` (Node.js), `pip` (Python), `maven` (Java).

Exemplo rápido (Node.js)
1. `node -v` / `npm -v` para checar versões.
2. `npm init -y` para criar `package.json`.
3. `npm install express --save` para adicionar dependência.

Dicas
- Use virtualenv/venv para isolar dependências Python.
- Configure `editorconfig` e linters para padronizar estilo.
