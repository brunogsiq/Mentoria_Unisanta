# 08.2 - Setup Local e Ferramentas

Checklist de setup local típico:

- Instalar runtime (Node/Python/Java) e gerenciador de pacotes.
- Configurar editor (VS Code) e extensões úteis.
- Configurar variáveis de ambiente e arquivo de exemplo `.env.example`.

Exercício: criar um script `setup.sh`/`setup.ps1` que prepara o ambiente de desenvolvimento.
