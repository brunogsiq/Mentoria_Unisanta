# 19 - Automação de Testes

Princípios e ferramentas para automatizar validações.

Arquitetura de testes
- Page Objects: separar lógica de interação da lógica de verificação.
- Comandos reutilizáveis: facilitar escrita de cenários.

Ferramentas por camada
- Web E2E: Cypress, Playwright, Selenium.
- API: Postman, RestAssured.
- Mobile: Appium, WebdriverIO.

Exemplo (estrutura simples)
- `tests/` com `pages/` (Page Objects) e `specs/` (cenários).

Dica
- Mantenha testes automatizados estáveis: isole dependências (mocks) quando possível.
