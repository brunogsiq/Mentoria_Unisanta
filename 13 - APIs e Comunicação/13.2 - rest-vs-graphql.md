# 13.2 - REST vs GraphQL

Comparação breve entre paradigmas de API:

- REST: recursos, verbos HTTP, cachê e simplicidade.
- GraphQL: consulta flexível, evita over/under-fetching, requer schema.

Exercício: crie uma query GraphQL simples e compare com equivalente REST.
