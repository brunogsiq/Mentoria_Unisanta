# 14 - Banco de Dados

Conceitos básicos e exemplos práticos para começar.

Modelos
- Relacional (SQL): tabelas e relacionamentos (MySQL, PostgreSQL).
- Não-relacional (NoSQL): documentos, chave-valor (MongoDB, Redis).

Exemplo SQL (seleção simples)
- `SELECT id, name FROM products WHERE price > 100;`

JOIN básico
- `SELECT o.id, u.name FROM orders o JOIN users u ON o.user_id = u.id;`

Dica
- Normalize dados inicialmente; otimize (index) quando identificar gargalos.
