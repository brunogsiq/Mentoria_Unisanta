# 27 - Fundamentos de Programação

Noções essenciais para começar a programar, com exemplos simples.

Conceitos básicos
- Variáveis: armazenam valores (`x = 10`).
- Tipos: números, strings, booleanos.
- Condicionais: `if/else` para decisões.
- Loops: `for`/`while` para repetir ações.

Exemplo (pseudo-JS)
- `if (age >= 18) { console.log('maior'); } else { console.log('menor'); }`
- `for (let i=0; i<5; i++) { console.log(i); }`
