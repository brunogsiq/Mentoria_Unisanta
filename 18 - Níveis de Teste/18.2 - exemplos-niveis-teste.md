# 18.2 - Exemplos de Níveis de Teste

Breve lista de exemplos por nível:

- Unit: testar funções puras e classes isoladas.
- Integration: testar camada de serviço com banco em memória.
- System/E2E: testar fluxo de usuário com ambiente controlado.

Exercício: criar um teste unitário e um teste de integração que usem o mesmo conjunto de dados.
