# 29 - DevOps

Visão prática sobre cultura e práticas DevOps.

Conceitos
- Integração entre desenvolvimento e operações para entregar software de forma contínua.
- Pipeline: automação de build, testes e deploy.

Práticas comuns
- Infra as Code (IaC): declarar infraestrutura (Terraform, Bicep).
- Monitoramento e feedback rápido após deploy.

Exemplo simples
- Pipeline que roda linter → testes → build → publica em staging para revisão.
