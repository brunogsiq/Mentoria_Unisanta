# 05 - Gerenciamento de Projetos

Objetivo: práticas e ferramentas para planejar e acompanhar entregas.

Termos-chave
- Epic: grande bloco de trabalho (várias features).
- Feature: funcionalidade maior dentro de uma epic.
- User Story: item de trabalho pequeno, com critérios de aceite.

Ferramentas comuns
- Jira / Azure DevOps: gerenciamento de backlog, boards e sprints.

Estimativas e métricas
- Story Points: estimativa relativa de esforço.
- Velocity: soma de story points entregues por sprint (ajuda no planejamento).

Exemplo prático
- Criar uma user story "Cadastro de produto" com critérios: campos obrigatórios, validações e teste E2E. Estimar 3 story points.

Dicas rápidas
- Separe planejamento e refinamento: refine histórias antes do planning.
- Use pequenos PRs e integração contínua para reduzir risco.
