# 04.2 - Kanban

Resumo do Kanban e práticas recomendadas:

- Principais conceitos: fluxo contínuo, limite de WIP, quadro Kanban.
- Práticas: visualizar trabalho, limitar WIP, gerir fluxo, políticas explícitas.
- Exemplos: cartões para tarefas, colunas (To Do, Doing, Done), políticas de definição de pronto.

Exercício prático: crie um quadro Kanban para uma sprint e defina limites de WIP para as colunas.
