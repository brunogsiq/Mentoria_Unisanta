# 01.2 - Exemplos Práticos

Breve exercício prático para fixar conceitos de Introdução à Tecnologia:

- Objetivo: criar uma API CRUD simples (Node.js/Express ou Python/Flask).
- Passos:
	1. Inicializar projeto (`npm init` / `pipenv`).
	2. Criar rota `GET /items` e `POST /items`.
	3. Persistir em arquivo JSON ou banco leve (SQLite).
	4. Escrever 2 testes automáticos (unit).

Recursos e sugestões de leitura:
- Node.js + Express: documentação oficial.
- Python + Flask: tutorial rápido.

Tarefas sugeridas:
- Implementar o endpoint `POST /items` e testar manualmente com `curl` ou Postman.
- Commitar mudanças e abrir PR com descrição clara.
