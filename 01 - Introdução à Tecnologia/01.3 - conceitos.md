# 01.3 - Conceitos Rápidos

Resumo de conceitos fundamentais mencionados na introdução:

- **API**: ponto de comunicação entre frontend e backend. Ex.: `GET /products` retorna lista de produtos.
- **CRUD**: Create, Read, Update, Delete — operações básicas sobre dados.

Exemplos rápidos:
- `POST /items` → cria um item
- `GET /items/1` → lê o item com id 1
- `PUT /items/1` → atualiza o item 1
- `DELETE /items/1` → remove o item 1
