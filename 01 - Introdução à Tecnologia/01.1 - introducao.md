# 01 - Introdução à Tecnologia

Panorama prático: visão geral de como um sistema funciona e fluxo básico entre frontend, backend e banco de dados.

Exemplo simples (fluxo):
1. Usuário clica em "Entrar" no app (frontend).
2. O frontend envia uma requisição ao backend: `POST /login` com email/senha.
3. Backend valida as credenciais no banco de dados.
4. Se válido, backend retorna um token ao frontend; o usuário entra no app.

Este arquivo mantém a visão geral; separei conceitos, testes, metodologias, ferramentas e próximos passos em arquivos individuais para facilitar estudos dirigidos.
