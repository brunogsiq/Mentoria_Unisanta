# 22 - Segurança Básica

Noções práticas de segurança para aplicações web.

Conceitos
- Autenticação: provar identidade (login).
- Autorização: controlar acessos (permissões).
- Token (JWT): forma comum de transmitir identidade entre frontend/backend.

Boas práticas rápidas
- Nunca guarde senhas em texto; use hashing (bcrypt, Argon2).
- Valide e sanitize entradas para evitar injeção (SQL, XSS).
- Gerencie segredos com serviços seguros (KeyVault, AWS Secrets Manager).

Regulação
- LGPD: cuidado com dados pessoais; minimize coleta e documente consentimentos.
