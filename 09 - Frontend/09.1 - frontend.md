# 09 - Frontend

Introdução prática ao desenvolvimento de interfaces web.

Tecnologias principais
- HTML/CSS: estrutura e estilo.
- JavaScript: interação e lógica no cliente.
- Frameworks: React, Angular, Vue (facilitam organização e reuso).

Exemplo rápido (consumir API)
- `fetch('/api/products').then(res => res.json()).then(data => render(data))`

Boas práticas
- Separar responsabilidade (componentes pequenos).
- Testes: unitários (Jest) e end-to-end (Cypress).

Dica
- Comece com um protótipo estático (HTML/CSS) antes de integrar com backend.
