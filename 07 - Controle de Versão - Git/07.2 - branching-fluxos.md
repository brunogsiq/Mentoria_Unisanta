# 07.2 - Branching e Fluxos de Trabalho

Visão dos principais fluxos de branching:

- Git Flow: uso de ramas `develop` e `release`.
- Trunk Based Development: integração frequente no `main`.
- Feature branches + PR: como criar PRs claras e revisáveis.

Dicas práticas: escreva mensagens de commit claras; rebase vs merge (quando usar cada um).
